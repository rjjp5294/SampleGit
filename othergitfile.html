Git Commands

  1. git clone - clones repository from github to my machine
  2. git status - shows what's different between what is on my machine and what is actually on github
  3. git add - add any new changes or files yet to be on github repository
  4. git commit - commits changes to github
  5. git push - finally, pushes commits up to github
  6. git pull - pulls files from correlating directory from github to your machine
